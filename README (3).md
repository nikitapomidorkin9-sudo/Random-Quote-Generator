#  Random Quote Generator (Генератор случайных цитат)

##  Автор
Онучин Никита 
## Описание
Графическое приложение для генерации вдохновляющих цитат. Позволяет случайным образом выбирать цитату из предустановленного списка, сохранять историю всех сгенерированных цитат, добавлять собственные цитаты и фильтровать историю по автору или теме. Все данные (список цитат и история) автоматически сохраняются в JSON-файлы, что обеспечивает их сохранность между запусками.

##  Как использовать
1. Убедитесь, что установлен **Python 3.8+**.
2. Скачайте или клонируйте репозиторий.
3. Запустите в приложение проиграмму Random Quote Generator.
4. В интерфейсе:
   - Добавление цитаты: заполните поля «Текст», «Автор», «Тема» → нажмите «Добавить цитату».
   - Генерация: нажмите «Сгенерировать случайную цитату» – цитата появится на панели справа и добавится в историю.
   - Фильтрация истории: выберите автора и/или тему в выпадающих списках – список истории обновится автоматически.

##  Примеры использования (Тест-кейсы)
|Действие|Ожидаемый результат|
|----------|--------------------|
|Оставить поля «Текст» или «Автор» пустыми и нажать «Добавить цитату» | Ошибка: "Текст цитаты не может быть пустым" или "Автор не может быть пустым".|
|Добавить цитату: "Программирование — это искусство", "Алан Кэй", "IT" | Цитата добавляется в общий список, становится доступна для генерации.|
|Нажать «Сгенерировать случайную цитату» | В панели генерации отображается случайная цитата, она добавляется в историю снизу.|
|Выбрать в фильтре «Автор» = Стив Джобс | В истории остаются только цитаты Стива Джобса.|
|Выбрать в фильтре «Тема» = Мотивация | Показываются только цитаты с темой «Мотивация».|
|Закрыть приложение и запустить снова | Ранее сгенерированные цитаты загружаются в историю, добавленные пользователем цитаты доступны для генерации.|
